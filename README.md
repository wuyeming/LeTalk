# LeTalk-智能语音音响
基于ESP32S3设计的一款智能语音音响<br> 
（目前固件使用的是虾哥的小智ai）
***
# 1.注意事项
屏幕连接方式使用的是12p的焊接，支持四线spi接口，驱动芯片为`ST7789V`<br>
